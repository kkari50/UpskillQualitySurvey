# Daily Sessions Resources

This folder contains downloadable resources for the Daily Sessions category questions.

## Resources

| File | Related Question | Description |
|------|------------------|-------------|
| `get-ready-checklist.pdf` | Q1: Area organization and materials | Self-checklist for BTs/RBTs to ensure session readiness |

## How to Add Resources

1. Name files using kebab-case (e.g., `get-ready-checklist.pdf`)
2. Supported formats: PDF, PNG, JPG
3. Update this README when adding new resources
4. Files are accessible at `/resources/daily-sessions/<filename>`

## Question Reference

1. Area is organized and necessary materials are readily available on a consistent basis.
2. Session begins within 5 minutes of the scheduled start time.
3. Sessions include teaching across behavior analytic modalities.
4. Clearly defined structure (visual schedules, pairing, etc.).
5. Reinforcers identified and used appropriately.
6. Breaks provided with appropriate activities.
7. Response to problem behavior is consistent across team members.
